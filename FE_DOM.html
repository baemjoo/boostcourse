DOM Node 조작하기

  • DOM, 즉 HTML은 동적으로 다양하게 변경될 수 있습니다.
  • DOM APIs 에서는 다양한 메소드와 속성을 지원하고 있습니다.
  • DOM API느 라이브러리를 쓰는 것 보다 대체로 더 빠릅니다.(그 차이가 크진 않습니다.)

DOM 탐색 APIs

몇가지 유용한 속성

  • tagName : 엘리먼트의 태그명 반환
  • textContent : 노드의 텍스트 내용을 설정하거나 반환
  • nodeType : 노드의 노드 유형을 숫자로 반환

사용 예제

  • ex) class가 edwith-table일 경우
    • var a = document.querySelector(".edwith-table");
    • a.textContent; //해당 element가 가지는 text node 출력
    • a.tagName; //해당 element의 태그 내용의 속성이 출력, 예시는 "TABLE" 출력
    • a.innerHTML //해당 element의 HTML 하위에 있는 element들이 HTML 문자열로 출력
    • a.firstElementChild //첫번째 자식이 출력
    • a.remove(); //해당 element가 삭제됨(브라우저 지원이 낮다.)

이동

  • childNodes : 엘리먼트의 자식 노드의 노드 리스트 반환(텍스트 노드, 주석 노드 포함)
  • firstChild : 엘리먼트의 첫 번째 자식 노드를 반환(firstChild의 공백이나 텍스트가 있으면 그것을 첫 번째 자식이라고 인식.)
  • firstElementChild : 첫 번째 자식 엘리먼트를 반환 (firstChild보다 잘 쓰인다.)
  • parentElement : 엘리먼트의 부모 엘리먼트 반환
  • nextSibling : 동일한 노드 트리 레벨에서 다음 노드를 반환
  • nextElementSibling : 동일한 노드 트리 레벨에서 다음 엘리먼트 반환

사용 예제

    
  <body>
    <div>
      <ul>

        </ul>
    </div>
  </body>

DOM 조작 API

  • removeChild() : 엘리먼트의 자식 노드 제거
  • appendChild() : 마지막 자식 노드로 자식 노드를 엘리먼트에 추가
  • insertBefore() : 기존 자식 노드 앞에 새 자식 노드를 추가
  • cloneNode() : 노드 복제
  • replaceChild() : 엘리먼트의 자식 노드 바꿈
  • closest() : 상위로 올라가면서 가장 가까운 엘리먼트를 반환

HTML을 문자열로 처리해주는 DOM 속성/메소드

  • innerText : 지정된 노드와 모든 자손의 텍스트 내용을 설정하거나 반환
  • innerHTML : 지정된 엘리먼트의 내부 html을 설정하거나 반환
  • insertAdjacenHTML() : HTML로 텍스트를 지정된 위치에 삽입

사용 예제

  • innerText 예제
  • 해당 노드의 텍스트 내용을 바꿉니다.
  • innerHTML 예제
  • 해당 엘리먼트의 html내용을 바꿉니다.
  • insertAdjacenHTML() 예제
  • 해당 위치에 텍스트를 삽입합니다.
