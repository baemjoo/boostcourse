JDBC (Java Database Connectivity)

  • 자바를 이용한 데이터베이스 접속과 SQL문장의 실행, 그리고 실행 결과로 얻어진 데이터의 핸들링을 제공하는 방법과 절차에 관한 규약
  • 자바 프로그램내에서 SQL문을 실행하기 위한 자바 API
  • SQL과 프로그래밍 언어의 통합 접근 중 한 형태

  • JAVA는 표준 인터페이스인 JDBC API를 제공

  • 데이터베이스 벤더 또는 기타 써드파티에서는 JDBC 인터페이스를 구현한 드라이버(driver)를 제공한다.

JDBC를 이용한 프로그래밍 방법

  • import java.sql.*
  • 드라이버를 로드한다.
    • Class.forName("com.mysql.jdbc.Driver");
  • Connection 객체를 생성
    • String dburl = "jdbc:mysql://localhost/dbName";
    • Connection con = DriverManager.getConnection(dburl,ID,PASSWORD)
    • Connection 객체를 생성하는 것 -> DB에 접속됐을 때 얻을 수 있는 객체
    • 즉, DB에 접속하는 부분
  • Statement 객체를 생성 및 질의 수행
    • Statement stme= con.cresteStatement();
    • 쿼리 문을 생성하는 것, 또 실행하는 것
    • Statement도 인터페이스
  • SQL문에 결과물이 있다면 ResultSet 객체를 생성
    • ResultSet rs = stmt.
      • .execute("query") //any SQL
      • .executeQuery("query") //SELECT
      • .executeUpdate("query") //INSERT,UPDATE,DELETE
    • query문의 결과를 ResultSet으로부터 시작해서 하나씩 꺼내옵니다.
    • Next() 메소드를 통해 결과 값이 있다면 true/ 없다면 false
  • 모든 객체를 닫는다. close()
    • rs.close();
    • stmt.close();
    • con.close();
    • 생성 순서의 역순으로 닫는다.

JDBC 클래스의 생성 단계

  1. DriverManager를 이용하여 Connection 인스턴스를 얻는다.
  2. Connection을 통해서 Statement를 얻는다.
  3. Statement를 이용해 ResultSet을 얻는다.
  4. 열었던 반대 순서로 닫아준다.
