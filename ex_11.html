position



  1. position

    2. position 속성으로 특별한 배치를 할 수도 있습니다.

    position 속성은 기본 static이다. 그냥 순서대로 배치됩니다.

  2. absolute

    3. absolute 속성은 기준점에 따라서 특별한 위치에 위치합니다.

    top/left/right/bottom 으로 설정합니다.

    기준점을 상위엘리먼트로 단계적으로 찾아가는데 static 이 아닌 position이 기준점입니다.

  3. relative

    4. 원래 자신이 위치해야 할 곳을 기준으로 이동합니다.

    top/left/right/bottom 으로 설정합니다.

  4. fixed

    5. viewport(전체화면) 좌측, 맨 위를 기준으로 동작합니다.
